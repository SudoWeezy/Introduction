>Following commands will be executed behind the scene:

```shell
pipx install algokit
git clone https://github.com/algorandfoundation/algokit-utils-py.git
cd algokit-utils-py
pip install .
cd ..
```

### Verify installation
1. `$ algokit -version`
2. `$ gh codespace ports`
